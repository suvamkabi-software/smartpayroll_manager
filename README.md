<h1 align="center">💼 SmartPayroll Manager</h1>

<p>
<b>SmartPayroll Manager</b> is a payroll management system designed to manage employee salary information.
The project demonstrates important <b>Object-Oriented Programming (OOP)</b> concepts such as
<b>abstraction, inheritance, encapsulation, and polymorphism</b>.
It also features a <b>Graphical User Interface (GUI)</b> that provides a simple and user-friendly way to add employees, calculate salaries, and manage payroll records.
</p>

<hr>
<p>
The system allows users to add employees, calculate salaries, remove employees, and display employee details.
This project was created for <b>learning and practice purposes</b>.
</p>

<hr>

<h2>🚀 Features</h2>

<ul>
<li>➕ Add Employee</li>
<li>💰 Calculate Employee Salary</li>
<li>📋 Display Employee Details</li>
<li>❌ Remove Employee</li>
<li>🧾 Manage Payroll Records</li>
</ul>

<hr>

<h2>🛠 Technologies Used</h2>

<ul>

<li>Object-Oriented Programming (OOP)</li>
<li>Java Collections (ArrayList)</li>
<li>Console / GUI Interface</li>
</ul>
